# 🐼 python-chronicles: Pandas Mastery

A professional, structured documentation of mastering the *Pandas* library—from foundational data structures to advanced performance optimization and real-world analytical workflows.

---

## 🏗️ Core Curriculum

### 🟢 Phase 1: The Foundation
- [x] *Pandas Basics:* Series vs. DataFrames, anatomy of objects.
- [x] *Inspection:* head(), tail(), info(), describe().
- [x] *File I/O:* Optimized ingestion from CSV, Excel, and JSON.

### 🟡 Phase 2: Selection & Cleaning
- [x] *Indexing:* Mastering .loc[] and .iloc[].
- [x] *Filtering:* Boolean indexing and multiple conditions.
- [x] *Data Integrity:* isnull, dropna, fillna, and astype.

### 🟠 Phase 3: Manipulation & Aggregation
- [x] *Modification:* Adding/deleting columns and sorting.
- [x] *Functional Mapping:* Using .apply() and .map().
- [x] *Grouping:* groupby() operations and .agg() functions.
- [x] *Relational:* SQL-style joins and concatenation.

### 🔴 Phase 4: Advanced Analytics
- [x] *Time Series:* to_datetime(), resampling, and rolling windows.
- [ ] *Reshaping:* Pivot tables, melt, stack/unstack.
- [ ] *Optimization:* Vectorization and memory management.

## 🛠️ Installation & Setup

Ensure you have a Python environment ready. It is recommended to use a virtual environment.
```bash
# Clone the repository
git clone https://github.com/yourusername/python-chronicles.git

# Navigate to the folder
cd python-chronicles

# Install dependencies
pip install pandas numpy openpyxl
```
